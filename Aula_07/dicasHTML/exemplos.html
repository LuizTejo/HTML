Exemplos HTML

O que são tags?

Como dissemos, o HTML é uma linguagem baseada em marcação.

Nós marcamos os elementos para mostrar quais informações a página exibe. Por exemplo, um título importante. Aquele título do artigo, da manchete do site, nós marcamos com uma tag/elemento chamado H1.

  • ‹li›‹li›
  • ‹p›‹p›
  • ‹src=”nome-da-imagem.jpeg” alt=”Texto alternativo caso a imagem não possa ser exibida”›

Exemplo do código-fonte

O código é simples de ser entendido. Os elementos são auto-explicativos:

‹!DOCTYPE html›

 ‹html›

   ‹head›

     ‹title›Primeiro documento em HTML‹/title›

   ‹/head›

   ‹body›

     ‹h1›Titulo da página‹/h1›

     ‹p›Este é o primeiro título.‹/p›

     ‹h2›Subtítulo da página‹/h2›

     ‹p›Este é o subtítulo.‹/p›

   ‹/body›

 ‹/html›

No código acima, nós temos a descrição de uma página simples. As Tags ‹head› e ‹/head› servem apenas para marcar a formatação do cabeçalho. Dentro desta tag, está inclusa as tags ‹title› e ‹/title›, que servem para incluir o título da página, normalmente localizada na aba do navegador.

Após a marcação ‹head›, temos o início da marcação ‹body› e ‹/body›. Esta marcação se refere ao corpo da página.

Nesta instrução, encontramos o título ‹h1› e ‹/h1›, que está presente o texto “Título da página”. Este título é o principal da página, que além de visual, também é utilizado por sistemas de busca para saberem do que se trata seu conteúdo.

A tag ‹p› e ‹/p›, servem para sinalizar um parágrafo.

As tags ‹h2›‹/h2› servem também para marcar um título da página. Neste caso, o título está abaixo hirarquicamente do título marcado como h1.
