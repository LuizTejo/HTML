Tudo sobre HTML

HTML é uma das linguagens que utilizamos para desenvolver websites. O acrônimo vem do inglês e significa Hypertext Markup Language ou em português Linguagem de Marcação de Hipertexto.


Trata-se de um padrão de marcação criado para páginas para Internet. Resumidamente, através dela é possível:

  • Descreve uma estrutura para ser lida por navegadores
  • Desenvolver sites
  • Criar conteúdo para ser acessível por toda Internet
  • Desenvolver aplicativos através de frameworks

Neste artigo, vamos falar basicamente tudo o que você precisa saber sobre HTML. Além disso, você poderá encontrar diversas referências sobre esta linguagem de marcação neste site.


Afinal, o que é HTML?

HTML é o código que é usado para estruturar uma página web e seu conteúdo.

Por exemplo, o conteúdo pode ser estruturado dentro de um conjunto de parágrafos, de uma lista de itens, ou usando imagens e tabelas de dados. Este artigo lhe dará uma compreensão básica da linguagem e suas funções.

HTML é muito usado para criar páginas que são exibidas através de um navegador. Esta página, por exemplo, contém HTML que são interpretados através do seu navegador.

Páginas de HTML são normalmente armazenadas em arquivos que usam as extensões .htm ou html. Um site pode incluir centenas ou mesmo milhares desses arquivos HTML mantidos em vários diretórios.

Seu servidor envia seus arquivos HTML para seu navegador quando você visita uma página web. Desta forma, seu navegador então lê os arquivos e os exibe. Algumas aplicações web não utilizam HTML estático, mas o geram em resposta a ações específicas em seus servidores.

Você também pode adicionar outros tipos de conteúdo, tais como arquivos CSS e JavaScript, imagens e vídeos, para adicionar mais recursos a uma página da web.
